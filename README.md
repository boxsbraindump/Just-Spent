就记一下

一个基于 Web 的极简主义记账工具，专为移动端单手操作设计。通过流畅的滑动交互和直观的图标，让你在几秒钟内完成一笔记录。

✨ 核心特性

滑动交互：左右滑动切换消费类别，灵感来自原生 App 的丝滑感。

多币种支持：支持 CNY (¥) 和 USD ($) 切换。

本地存储：数据完全保存在浏览器的 localStorage 中，无需注册，保护隐私。

可视化统计：

今日支出：实时显示当日消费总额。

月度分析：自动汇总本月消费，并显示各分类占比进度条。

年度总结：一目了然全年的消费积累。

响应式设计：完美适配所有手机屏幕尺寸，支持双端（iOS/Android）PWA 体验。

🚀 快速开始

本项目是一个单文件 (Single-file) 应用，无需任何构建步骤。

下载或克隆仓库。

直接在浏览器中打开 index.html 即可使用。

🌐 部署到 GitHub Pages

由于项目是纯静态的，你可以非常轻松地将其部署为在线 App：

将 index.html 上传到你的 GitHub 仓库。

进入仓库的 Settings -> Pages。

在 Branch 选项中选择 main (或 master) 分支。

点击 Save，稍等片刻即可通过 GitHub 提供的链接在线访问。

🛠️ 技术栈

HTML5/CSS3：结构与自定义动画。

Tailwind CSS：现代化的 UI 样式框架。

Vanilla JavaScript：纯原生逻辑控制，无外部依赖。

LocalStorage API：持久化用户数据。

📝 隐私声明

你的所有财务数据均不会上传到任何服务器。数据仅存储在你当前使用的设备浏览器缓存中。清空浏览器缓存或点击应用内的“清空”按钮将导致数据抹除，建议定期截图备份重要数据。
